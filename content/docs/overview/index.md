Hugo Narrow is a clean and modern Hugo theme for personal blogs, built with Tailwind CSS 4.0, featuring multiple color schemes and user customization support.

## Features

- Multiple Color Schemes
- Custom Colors
- Dark Mode
- Table of Contents
- Search
- Comments
- Analytics
- Enhanced Markdown
- Enhanced Code Blocks
- Multilingual
- Katex
- Mermaid
- Lightbox by GLightbox
- Justified Gallery with markdown
- Masonry Gallery with shortcode
