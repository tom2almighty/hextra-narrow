This section describes the feature configurations of Hugo Narrow. All these configurations are located under the `params` node in the `hugo.yaml` configuration file, covering the settings of site homepage, navigation bar, Dock bar, table of contents, comments, image gallery and other functions. It serves as the core functional configuration module of Hugo Narrow.
