Hugo Narrow supports three import methods:

- Hugo Mod
- Git Submodule
- Download zip files

The Hugo Mod method is fast and convenient. If you need advanced customization of the theme, the Git Submodule method is more suitable after fork. The method of downloading the theme compression package is also relatively convenient, but it is difficult to update the theme subsequently, so this method is not recommended.
