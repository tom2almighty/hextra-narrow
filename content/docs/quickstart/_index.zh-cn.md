Hugo Narrow 支持三种方式导入：

- Hugo 子模块
- Git 子模块
- 下载主题压缩包

Hugo 模块方式快速方便，如果需要对主题进行高级自定义，更适合 fork 后使用 git 子模块方式导入。下载主题压缩包的方式也较为方便，但后续更新主题困难，不推荐使用此方式。
